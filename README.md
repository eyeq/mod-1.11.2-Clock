# [置き時計]Clock

[**ダウンロード**](https://github.com/eyeq/mod-1.11.2-Clock/releases/download/1.0/1.11.2-Clock-1.0.jar)

時計ブロックを追加します。  
時間単位で表示が変化し、その時間に相当するレッドストーン信号を出力します。  
右クリックすると、ワールドの時間が1時間進みます。
